## Verification Steps

1. **Run the dev server**: `npm run dev` — verify the app loads
2. **Complete full flow**: Splash → Onboarding → Profile → Home → Lesson 1 → Comic → Activities → Results
3. **Verify all 12 comic pages**: Navigate through each and check dialog, chalkboard, quiz
4. **Test audio**: Tap chalkboard words, use Read Aloud button, check slow speed setting
5. **Test activities**: Complete all 5 activity types across all 6 lessons
6. **Test wrong answers**: Verify hints appear and retry is supportive
7. **Check teacher dashboard**: View skill analysis, lesson progress, badges
8. **Test responsiveness**: Check on mobile viewport (375px), tablet (768px), desktop
9. **Test dark mode**: Toggle theme in settings and verify all screens
10. **Test text sizes**: Toggle small/medium/large and verify readability
11. **Test keyboard navigation**: Use arrow keys and space in comic reader
12. **Test swipe navigation**: Swipe left/right in comic reader on touch device
